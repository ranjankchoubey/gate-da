## Q301

[MCQ]

Which of the following is a valid PMF for a discrete random variable $X$ taking values $\{1, 2, 3\}$?

- **(A)** $P(X=1)=0.3,\ P(X=2)=0.3,\ P(X=3)=0.3$
- **(B)** $P(X=1)=0.5,\ P(X=2)=0.3,\ P(X=3)=0.2$
- **(C)** $P(X=1)=-0.1,\ P(X=2)=0.6,\ P(X=3)=0.5$
- **(D)** $P(X=1)=0.4,\ P(X=2)=0.4,\ P(X=3)=0.4$

**Answer:** B

---

## Q302

[NAT]

A discrete random variable $X$ has PMF: $P(X=1)=0.2$, $P(X=2)=0.3$, $P(X=3)=k$, $P(X=4)=0.1$. For this to be a valid PMF, what is the value of $k$?

**Answer:** 0.4

## Q303

[MSQ]

Which of the following conditions must hold for $p(x) = P(X = x)$ to be a valid PMF? (Select all that apply)

- **(A)** $p(x) \geq 0$ for all $x$
- **(B)** $\sum_x p(x) = 1$
- **(C)** $p(x) \leq 1$ for all $x$
- **(D)** $p(x)$ must be strictly positive for every real number $x$

**Answer:** A, B, C

## Q304

[NAT]

A random variable $X$ has the following PMF:

| $x$ | 0 | 1 | 2 | 3 |
|-----|---|---|---|---|
| $P(X=x)$ | 0.1 | 0.3 | 0.4 | 0.2 |

What is $E[X]$?

**Answer:** 1.7

## Q305

[MCQ]

For a random variable $X$, $E[X] = 3$ and $E[X^2] = 13$. What is $\text{Var}(X)$?

- **(A)** 4
- **(B)** 10
- **(C)** 13
- **(D)** 16

**Answer:** A

## Q307

[MCQ]

For a random variable $X$ with $E[X] = \mu$ and $\text{Var}(X) = \sigma^2$, what is $\text{Var}(aX + b)$?

- **(A)** $a^2 \sigma^2 + b^2$
- **(B)** $a^2 \sigma^2$
- **(C)** $a\sigma^2 + b$
- **(D)** $\sigma^2$

**Answer:** B

## Q308

[MCQ]

The CDF of a discrete random variable $X$ is defined as:

- **(A)** $F(x) = P(X = x)$
- **(B)** $F(x) = P(X \geq x)$
- **(C)** $F(x) = P(X \leq x)$
- **(D)** $F(x) = 1 - P(X = x)$

**Answer:** C
